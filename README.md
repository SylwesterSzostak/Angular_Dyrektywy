To jest cała zawartość folderu src. Folder images należy wrzucić do folderu public - wtedy obrazy będą wczytane poprawnie.
Po stworzeniu czystego projektu należy tylko skopiować pliki z katalogu src oraz z podkatalogu app - jeżeli zystem poprosi o zamianę to trzeba zezwolić.
Projekt jest teraz czytelniejszy i bardziej zrozumiały.
Co zostało zmienine:
- edycji dokonujemy w pliku app.ts
- wszystkie style css są w pliku app.css, który jest dołączony do projektu.
- dla obiektu Osoba został stworzony interfejs(było w klasie drugiej), który również jest dołączony w pliku app.ts
- usunięte zostało łączenie dwukierunkowe oraz przycisk zmiany obrazka.
